# 本番環境 必要スペック試算

計測日: 2026-05-24  
計測環境: Intel Core i5-8500T @ 2.10GHz / 4vCPU / 4.8GB RAM (KVM VM)

## 計測値

| 処理 | 所要時間 | CPU使用率 |
|------|---------|----------|
| Whisper推論（baseモデル、無音） | 1.3秒 | 全コアの50〜100% |
| Whisper推論（発話あり） | 1.5〜2.5秒 | 同上 |
| OpenRouter API呼び出し | 2〜4秒 | ほぼ0（待ち） |
| アイドル（VAD待機中） | — | 2〜5% |
| edge-tts | 0.5秒 | 軽微 |

## 本番想定（寝室・静音環境）

- Whisper呼び出し頻度: 1〜3回/分（発話時のみ）
- 1回の会話セッション: 30〜60秒、Whisper 5〜10回
- 環境音が少ないためVAD誤検知がほぼなくアイドル時はほぼ無負荷

## 推奨スペック

| 項目 | 最低限 | 推奨 |
|------|--------|------|
| vCPU | 2コア | 4コア |
| RAM | 1GB | 2GB |
| ストレージ | 8GB | 16GB |

## 判定

**現在の4vCPU / 4.8GB RAM構成で本番運用十分。**

- Whisperが推論中は1〜2コアをほぼ専有（1.3秒間）するため、2コアだと推論中に他リクエストが詰まる可能性あり
- 常駐メモリ: uvicorn+Whisperモデルで約520MB、OS込みで1GB超
- 寝室の静音環境ならリビングと比べCPU負荷が大幅に下がり快適に動作する見込み
